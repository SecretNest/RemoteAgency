8 Advanced event

Let's refine the event support.

  • While raising a standard event, the first parameter should be the object of caller itself. But it may not be serialized to another site. Let's make a compromised way: keep the coding standard, but not transfer it through Remote Agency.

    Simply, mark the event with EventParameterIgnoredAttribute. Or, you can mark the parameter in delegate with ParameterIgnoredAttribute. ParameterIgnoredAttribute can also be support in method calling.

  • A non-standardized event could have different parameters and return. Remote Agency can support it also.

There are something else not covered in this example:

  • Two way property is also supported. If you can define it in delegate, ParameterTwoWayPropertyAttribute in Example 3 is a good way. Or, you can also use EventParameterTwoWayPropertyAttribute at event definition.

  • To optimized the event subscribing speed, you may want to check the EventSubscritionMode.

Event

  1. Define a service contract interface and the delegate.

    C#
    Copy
    public delegate string MyTestHandler(string value);

public interface IHello
{
  [EventParameterIgnored("sender")]
  event EventHandler WorldOpened;

  event MyTestHandler MyTestEvent;
  void HelloWorld();
}

  2. Create the service class.

    C#
    Copy
    class Hello : IHello
{
  public event EventHandler WorldOpened;
  public event MyTestHandler MyTestEvent;

  public void HelloWorld()
  {
    WorldOpened?.Invoke(this, EventArgs.Empty);

    string value = MyTestEvent?.Invoke("Request");
    if (value == "Response")
      Console.WriteLine("Pass.");
  }
}

  3. In Main method, define the service object, create a service wrapper and put it into server site.

    C#
    Copy
    Hello serviceObject = new Hello();
//Because we are using easy encapsulation, all creators, serializing helpers, packing helpers and entity code builders are encapsulated into just one call.
serverSite.AddServiceWrapper<IHello>(serviceObject, out Guid serviceWrapperInstanceId);

  4. Add methods for handling events for client.

    C#
    Copy
    private static void Proxy_WorldOpened(object sender, EventArgs e)
{
  Console.WriteLine("Hello World!");
}

private static string Proxy_MyTestEvent(string value)
{
  if (value == "Request")
    return "Response";
  else
    return "Unknown";
}

  5. In Main method, create a proxy object, put it into client site, and add the handler of the event.

    C#
    Copy
    IHello proxy = clientSite.AddProxy<IHello>(serviceWrapperInstanceId, out _);
proxy.WorldOpened += Proxy_WorldOpened;
proxy.MyTestEvent += Proxy_MyTestEvent;

  6. In Main method, call the method of proxy.

    C#
    Copy
    proxy.HelloWorld();

  7. Dispose objects after testing.

    C#
    Copy
    clientSite.RemoveManagingObject(proxyInstanceId, true);
serverSite.RemoveManagingObject(serviceWrapperInstanceId, true);
