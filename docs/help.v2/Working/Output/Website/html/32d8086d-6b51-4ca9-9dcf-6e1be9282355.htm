Version History

The topics in this section describe the various changes made to the Remote Agency over the life of the project.

Version History

  • Version: 2.0.0

    Caution: Contains multiple break changes.

    Merges serializer to main package.

    Changes to use IL directly from Roslyn compiling.

    Combines some attributes.

    Adds function: Thread lock: specify the running thread / task scheduler for running assets.

    Add support: Indexer

    Changes to use Json and binary serializer instead of Data Contract serializer.

    Changes the interfaces of Remote Agency.

    Changes the name of RemoteAgency.Base to RemoteAgency.Abstraction.

    Adds a RemoteAgency.Neat with all functions but built-in serializers and the reference of Sequential Scheduler.

    Removes .Net Standard from target due to limitation of System.Reflection.Emit.

    Removes assembly output, saving built assembly to file, in .Net Core due to limitation of System.Reflection.Emit.

  • Version 1.x

    Check Remote Agency Version1 Doc for details.
