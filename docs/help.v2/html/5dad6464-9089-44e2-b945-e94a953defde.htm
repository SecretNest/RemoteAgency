RemoteAgency.Abstraction

SecretNest.RemoteAgency.Abstraction is the library contains interfaces, attributes and exceptions (other than thrown while building) of Remote Agency.

This library is for the project which needs to work with, but not hosts a instance of Remote Agency, like a project contains services or an extended serializer. For the project referenced SecretNest.RemoteAgency or SecretNest.RemoteAgency.Neat, SecretNest.RemoteAgency.Abstraction is referenced automatically without manual operation required.
