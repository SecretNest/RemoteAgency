Remarks

When IsIgnored is set to , the return value of the asset, if exists, will not be sent back the caller.

ReturnIgnoredAttribute affect the return value only. For ignoring from all responses, use AssetOneWayOperatingAttribute or PropertyGetOneWayOperatingAttribute instead.

When IsIgnored is set to , CustomizedPropertyGetValuePropertyNameAttribute and CustomizedReturnValueEntityPropertyNameAttribute on or in the same asset, and the delegate related to the asset if the asset is an event, will be ignored.

By specifying this on properties, only get operating will be affected.

The one marked on the event has higher priority than the one marked on the delegate of the same event.

The one marked on the return value has higher priority than the one marked on the same member (method, event or delegate).

Without AssetOneWayOperatingAttribute, PropertyGetOneWayOperatingAttribute or ReturnIgnoredAttribute specified, no return value is ignored.
