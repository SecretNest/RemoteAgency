RemoteAgency and RemoteAgency.Neat

SecretNest.RemoteAgency is the main library of Remote Agency.

Supported functions

  • Methods

    • Normal methods

    • Overloading

    • Generic methods

    • Parameters: params, out, ref and default value

    • With or without return

  • Properties

    • Method: get and set

    • Indexer

  • Events

    • Method: add, remove and raise

    • Parameters: params, out, ref and default value

    • With or without return

  • One-way calling

  • Specified task scheduler for calling asset

  • Specified field, property of parameter sending back to caller

  • Local exception redirecting

  • Asset, entity and property renaming

  • Asset, parameter ignoring

  • Assembly caching

  • Not in Neat

    • Built-in Json Serializer: Newtonsoft Json serializer and entity builder for it.

    • Built-in Binary Serializer: BinaryFormatter serializer shipped with CLR and entity builder for it.

    • Sequential scheduler: A task scheduler that run tasks on a single thread, referenced from Nuget.

Neat Version

For the user don't require built-in serializers and sequential scheduler, a neat version can be use instead.

When using neat version, CreateWithBinarySerializer, CreateWithJsonSerializer and TryAddSequentialScheduler are not included.
