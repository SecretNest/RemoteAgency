Binary Formatter Warning

BinaryFormatter is included in Binary Serializer, which is dangerous and should not be used.

BinaryFormatter

Microsoft has stated that BinaryFormatter is dangerous and is not recommended for data processing. Please check BinaryFormatter security guide (Microsoft Docs) for further information.

Use BinaryFormatter with Caution

Though it is not recommended, the BinaryFormatter is the fastest way in serializing and deserializing process. When it is necessary, uses it with caution and passes true as the first argument when calling CreateWithBinarySerializer or initializing RemoteAgencyBinarySerializer, or an exception will be thrown.
