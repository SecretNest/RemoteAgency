propertySettingTimeout
Type: SystemInt32
The length of time for waiting response for property setting, in milliseconds; or -1 to indicate that the waiting does not time out.
Remarks

This constructor is for setting the timeout for property getting and setting separately only. To set to the same value, or set for asset other than property, uses OperatingTimeoutTimeAttribute(Int32) instead.

To set the timeout for event adding, removing and raising separately, uses OperatingTimeoutTimeAttribute(Int32, Int32, Int32) instead.
