6: Indexer

Remote Agency supports indexer property.

Prerequisites
Before using this example code, an app need to be created first. Follow this doc to prepare that first.
Brief

Remote Agency supports indexer property, no matter the type and number of parameters.

A parameter which is marked with ParameterIgnoredAttribute will be ignored from passing to target and always has the default value. See Parameter Ignoring and Two-way for details.

Steps

  1. Create an interface that contains all the assets used for communication.

    C#
    VB
    Copy
    public interface ITest6
{
    int this[int index]
    {
        get;
        set;
    }

    [OperatingTimeoutTime(1000, 2000)]
    int this[string name, [ParameterIgnored]string ignored]
    {
        get;
        set;
    }
}
    Public Interface ITest6
    Default Property Item(index As Integer) As Integer

    <OperatingTimeoutTime(1000, 2000)>
    Default Property Item(name As String, <ParameterIgnored> ignored As String) As Integer
End Interface

  2. Create a server class that implement the interface created above.

    C#
    VB
    Copy
    public class Server6 : ITest6
{
    private readonly int[] _data = {101, 102, 103, 104, 105};

    public int this[int index]
    {
        get => _data[index];
        set => _data[index] = value;
    }

    public int this[string name, string ignored]
    {
        get
        {
            if (ignored != null)
            {
                Console.WriteLine("Server side: Why is this happened?");
            }
            if (int.TryParse(name, out var index))
            {
                return _data[index];
            }
            else
            {
                Thread.Sleep(5000);
                return -1;
            }
        }
        set
        {
            if (ignored != null)
            {
                Console.WriteLine("Server side: Why is this happened?");
            }
            if (int.TryParse(name, out var index))
            {
                _data[index] = value;
            }
            else
            {
                Thread.Sleep(5000);
            }
        }
    }
}
    Public Class Server6
    Implements ITest6

    Private ReadOnly _data As Integer() = {101, 102, 103, 104, 105}

    Default Public Property Item(index As Integer) As Integer Implements ITest6.Item
        Get
            Return _data(index)
        End Get
        Set(value As Integer)
            _data(index) = value
        End Set
    End Property

    Default Public Property Item(name As String, ignored As String) As Integer Implements ITest6.Item
        Get
            If ignored IsNot Nothing Then
                Console.WriteLine("Server side: Why is this happened?")
            End If
            Dim index As Integer
            If Integer.TryParse(name, index) Then
                Return _data(index)
            Else
                Thread.Sleep(5000)
                Return -1
            End If
        End Get
        Set(value As Integer)
            If ignored IsNot Nothing Then
                Console.WriteLine("Server side: Why is this happened?")
            End If
            Dim index As Integer
            If Integer.TryParse(name, index) Then
                _data(index) = value
            Else
                Thread.Sleep(5000)
            End If
        End Set
    End Property
End Class

  3. Place this code to a method, main method for example.

    C#
    VB
    Copy
    //test router
var router = new RemoteAgencyRouter<byte[], object>();

//Server
var originalService = new Server6();
using var serverRemoteAgencyInstance = RemoteAgencyBase.CreateWithBinarySerializer(true);
router.AddRemoteAgencyInstance(serverRemoteAgencyInstance);
var serverSiteId = serverRemoteAgencyInstance.SiteId;
var serviceWrapperInstanceId = serverRemoteAgencyInstance.CreateServiceWrapper(originalService);

//Client
using var clientRemoteAgencyInstance = RemoteAgencyBase.CreateWithBinarySerializer(true);
router.AddRemoteAgencyInstance(clientRemoteAgencyInstance);
var clientProxy = clientRemoteAgencyInstance.CreateProxy<ITest6>(serverSiteId, serviceWrapperInstanceId).ProxyGeneric;

//Run test
Console.WriteLine("this[2](Get, 102):");
Console.WriteLine(clientProxy[2]);

Console.WriteLine("this[4](Set+Get, No return):");
clientProxy[4] += 100;

Console.WriteLine("this[\"4\", \"brabrabra\"](Get, 204):");
Console.WriteLine(clientProxy["4", "brabrabra"]);

Console.WriteLine("this[\"Timeout\", \"brabrabra\"](Get, Timeout):");
try
{
    Console.WriteLine(clientProxy["Timeout", "brabrabra"]);
}
catch (Exception e)
{
    Console.WriteLine("Predicted Exception: " + e);
}

Console.WriteLine("this[\"Timeout\", \"brabrabra\"](Set, Timeout):");
try
{
    clientProxy["Timeout", "brabrabra"] = 0;
}
catch (Exception e)
{
    Console.WriteLine("Predicted Exception: " + e);
}

Console.Write("Press any key to continue...");
Console.ReadKey(true);
Console.WriteLine();

'Run test

Console.WriteLine(clientProxy("4", "brabrabra"))

Console.WriteLine("Item(""Timeout"", ""brabrabra"")(Get, Timeout):")
Try

    Console.WriteLine(clientProxy("Timeout", "brabrabra"))
Catch ex As Exception
    Console.WriteLine("Predicted Exception: " + ex.ToString())
End Try

Console.WriteLine("Item(""Timeout"", ""brabrabra"")(Set, Timeout):")

serverRemoteAgencyInstance.Dispose()
clientRemoteAgencyInstance.Dispose()

    When creating proxy, target site id and service wrapper instance id need to be specified.

    Site id is used to locate the server. In a real scenario, servers are usually deployed into different applications even computers. Message routing component, like Remote Hub, need to use the site id to deliver messages generated from proxy to the right server instance.

    Service wrapper instance id is used to locate the service object which contains user code. This is used by Remote Agency to deliver the message to the right service object registered by CreateServiceWrapper (or CreateServiceWrapper).

    No such id is required when creating service wrapper because the response messages are sent to the source of the request.
