Remarks

Use this attribute to mark an attribute at the same place in the created class.

When using the parameterless constructor of target attribute, attributeConstructorParameterTypes and attributeConstructorParameters are not required.

When using a constructor with all parameters specified in order, sets types of parameters of constructor to attributeConstructorParameterTypes and values to attributeConstructorParameters. When the length of attributeConstructorParameters is smaller than the length of attributeConstructorParameterTypes, the missing parameters will be filled by .

When named parameter is required for constructing, sets types of full form parameters of constructor to attributeConstructorParameterTypes and leading values to attributeConstructorParameters. Marks the rest one by one using AttributePassThroughIndexBasedParameterAttribute. The AttributeId should have the same value as attributeId.

When properties need to be set while initializing, marks the properties one by one using AttributePassThroughPropertyAttribute. The AttributeId should have the same value as attributeId.

When fields need to be set while initializing, marks the fields one by one using AttributePassThroughFieldAttribute. The AttributeId should have the same value as attributeId.
