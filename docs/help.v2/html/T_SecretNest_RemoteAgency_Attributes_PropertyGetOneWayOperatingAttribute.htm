Remarks

When IsOneWay is set to , the getting operating return the default value to the caller due to lack of response. Any exception raised from the user code on target site will not be transferred to the caller site.

By specifying this on properties, only getting operating is affected. If one way setting operating is expected, mark the property with AssetOneWayOperatingAttribute.

When this attribute is absent, the one way mode on property getting operating of this asset is disabled.
