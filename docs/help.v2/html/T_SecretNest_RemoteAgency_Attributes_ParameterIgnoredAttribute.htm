Remarks

EventParameterIgnoredAttribute can be marked on event, with higher priority than ParameterIgnoredAttribute with the same parameter.

When IsIgnored is set to , ParameterReturnRequiredAttribute, ParameterReturnRequiredPropertyAttribute, EventParameterReturnRequiredAttribute, EventParameterReturnRequiredPropertyAttribute, CustomizedEventParameterEntityPropertyNameAttribute and CustomizedParameterEntityPropertyNameAttribute on or in the same parameter of the asset and the delegate related to the asset if the asset is an event, will be ignored.

Without EventParameterIgnoredAttribute or ParameterIgnoredAttribute specified, no parameter is ignored.
