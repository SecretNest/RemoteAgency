Accessing Metadata of Message in Service Method

Metadata of message instance includes id of sender and target, id of instance and site, message type, message id, asset name and some other properties.

To check all properties, see MessageInstanceMetadata for details.

Query in service methods

In service mothods, inluding property methods and event handler methods, metadata of message instance can be queried by accessing CurrentMessageInstanceMetadata.

Note: This property is separated based on thread. Accessing it in dedicated thread or async task is not supported.

Use metadata for filtering messages

Metadata of message instance, as well as message body and other properties can be provided for filtering messages.

To use message filtering, see Message Filtering for details.
