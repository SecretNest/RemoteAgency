Remote Agency Base

SecretNest.RemoteAgency.Base is a library contains all base classes, interfaces, attributes of Remote Agency.

Use this library

A user of Remote Agency does not need to reference this library directly. This is handled while your project references SecretNest.RemoteAgency.

This library contains all components which are required to communicate between SecretNest.RemoteAgency and its data serializer solutions. This library is used to develop a data serializer solution library which is designed to work with Remote Agency.

  • Abstract classes:

    • EntityCodeBuilderBase

    • PackingHelperBaseTNetworkMessage, TSerialized

    • SerializingHelperBaseTSerialized, TEntityBase

  • Interfaces:

    • IAssemblyCacheOperatingsTSerialized, TEntityBase

    • ICommunicateTSerialized

  • Delegates:

    • LoadCachedAssemblyCallbackTSerialized, TEntityBase

    • SaveCachedAssemblyCallbackTSerialized, TEntityBase

  • Attributes

  • Exceptions:

    • AssetNotFoundException

    • WrappedException

  • Helper classes:
