Preparing

The things you need:

  • An editor that support developing class library.

  • Decide which class should be chosen as the base class of all entity.

    Object should be a good choice if there is no special requirement.

  • The data format of serialized data.

    Usually, string or a binary format, like byte array, is a good approach.

  • The data format of network message.

    Serialized data and message headers will be packaged into this format while transferring among different sites.

Coding

  1. Create a class library project;

  2. Reference NuGet package: SecretNest.RemoteAgency.Base;

  3. Create an inherited class based on EntityCodeBuilderBase;

    This class will be used to generate entity classes and the code for accessing them.

  4. Create an inherited class based on SerializingHelperBaseTSerialized, TEntityBase;

    This class will be used to serialize asset accessing to a message and reverse.

    This class must be set to public. The instances of this class will be initialized and used in generated classes also.

  5. Create an inherited class based on PackingHelperBaseTNetworkMessage, TSerialized;

    This class will be used to packing and unpacking message.

  6. (Optional) You may create a class like RemoteAgencyManagerEncapsulated to make user accessing easier.

You could use SecretNest.RemoteAgency.DataContractSerializer and SecretNest.RemoteAgency.DataContractSerializer.EasyEncapsulation as an example while writing your own code. Code is available at github.
