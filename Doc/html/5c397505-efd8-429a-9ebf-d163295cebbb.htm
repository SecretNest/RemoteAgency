Disposing Proxy

The disposing message of proxy object need to be transferred to the service wrapper. If the transportation of message is not reliable, this need to be handle by user code.

Proxy need disposing

On a reliable transportation, the disposing message of proxy object will be sent to service wrapper automatically. These code is only for remediation.

When a proxy object need disposing is found, DisposingMessageRequiredProxyAdded will be raised.

When a proxy object, which needs sending disposing message, is removed, DisposingMessageRequiredProxyRemoved will be raised.

When a proxy object is disposed, OnProxyDisposed should be called.

When a whole site is down, you can remove each proxy of it, or simply call OnProxiesOfSiteDisposed.

Call a method for a proxy which is disposed will be ignored.
